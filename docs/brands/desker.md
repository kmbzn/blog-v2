# 데스커 (DESKER)

데스커(DESKER)는 한국의 가구·인테리어 전문 기업 **퍼시스그룹(Fursys Group)** 산하의 홈 오피스·학습 가구 브랜드다. 2018년 론칭 이후 전동 스탠딩 데스크, 시스템 책상, 의자 등을 주력으로 하며, "일하는 공간을 설계한다"는 브랜드 철학 아래 국내 홈 오피스·재택근무 가구 시장을 주도하고 있다.

퍼시스그룹은 국내 1위 사무용 가구 기업으로 오피스 가구 브랜드 **퍼시스(FURSYS)**와 학생·홈 오피스용 브랜드 **일룸(ILOOM)**도 같은 그룹 산하에 있다. 데스커는 그중 성인 홈 오피스·작업 공간에 집중하는 포지션이다.

## 주요 제품 라인

### 전동 스탠딩 데스크

데스커의 대표 제품군. 전동 액추에이터를 이용해 버튼 하나로 앉기·서기 높이를 조절한다.

**주요 모델**

| 모델 | 특징 |
|------|------|
| 스탠딩 데스크 1200 | 1200×600mm 표준 사이즈, 단모터 |
| 스탠딩 데스크 1400/1600 | 넓은 작업 면적, 듀얼 모터 |
| L형 스탠딩 데스크 | 코너형, 듀얼 모니터·멀티 작업 환경에 적합 |
| 스탠딩 데스크 프레임 | 상판 별도 구매형, 원하는 상판에 조합 가능 |

전동 스탠딩 데스크의 높이 범위는 일반적으로 620~1270mm 수준이며, 4단계 메모리 설정이 가능한 컨트롤러를 제공한다.

### 시스템 책상 (책상 + 상부장 + 책장 조합)

학습·작업 공간을 효율적으로 구성하는 모듈형 시스템 가구. 책상 상판 위에 수납 선반·상부장을 결합해 책·서류·모니터를 정리한다.

- 책상 + 상부 오픈장 + 측면 수납 조합 구성
- 화이트, 그레이, 네이처 등 컬러 라인업
- 각 모듈 단독 구매 후 조합 가능

### 의자

**데스커 체어 라인**은 인체공학 설계에 집중한다.

| 모델 | 특징 |
|------|------|
| 에르고 체어 | 요추 지지, 4D 팔걸이, 메시 등받이 |
| 미듐 체어 | 중간 사이즈, 좁은 공간에 적합 |
| 팔걸이 없는 기본형 | 저가, 단순 작업용 |

### 액세서리

- **모니터 암 (Single/Dual)**: C클램프·그로멧 방식
- **케이블 트레이**: 책상 하부 케이블 정리
- **발 받침대**: 스탠딩 데스크 사용 시 발 피로 감소
- **모니터 선반**: 모니터 하단 공간 활용

## 스탠딩 데스크를 선택해야 하는 이유

장시간 앉아서 작업하는 환경에서 스탠딩 데스크는 자세 교정과 허리 피로 감소에 도움을 준다.

| 효과 | 설명 |
|------|------|
| 허리 부담 감소 | 서 있는 자세에서 허리 디스크 압력이 낮아짐 |
| 집중력 유지 | 자세 변화가 졸음·집중력 저하를 줄임 |
| 칼로리 소모 | 서 있을 때 앉아 있을 때보다 시간당 약 50kcal 추가 소모 |
| 목·어깨 경직 감소 | 높이 변경으로 모니터 시선 각도를 조정 |

단, 하루 종일 서 있는 것도 좋지 않다. **앉기·서기를 30~60분 간격으로 교대**하는 것이 이상적이다.

## 가격대

데스커 제품은 국내 홈 오피스 가구 중 **중~프리미엄** 구간에 위치한다.

| 제품군 | 가격대 |
|--------|--------|
| 기본 책상 | 15만원~ |
| 전동 스탠딩 데스크 1200 | 40~60만원 |
| 전동 스탠딩 데스크 1600 (듀얼모터) | 70~90만원 |
| L형 스탠딩 데스크 | 90~130만원 |
| 시스템 책상 (조합) | 50~120만원 |
| 에르고 체어 | 30~50만원 |

## 경쟁 브랜드 비교

| 브랜드 | 포지션 | 특징 |
|--------|--------|------|
| **데스커 (DESKER)** | 국내 중프리미엄 | 퍼시스그룹 품질, A/S 안정적 |
| FlexiSpot | 가성비 수입 | 중국산, 가격 저렴, 품질 편차 있음 |
| Uplift | 해외 프리미엄 | 미국 브랜드, 마감 최상위, 가격 높음 |
| 시디즈 | 국내 의자 전문 | 의자 중심, 책상도 있음 |
| 일룸 | 국내 학생용 | 학습·아동 가구 중심 |

A/S 측면에서 국내 기업 데스커는 부품 조달과 출장 수리가 용이하다는 장점이 있다.
