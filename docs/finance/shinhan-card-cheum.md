# 신한카드 처음

신한카드 처음은 2024년 4월 출시된 신한카드의 젊은 층(MZ세대) 타겟 신용카드다. 월정액·구독 서비스와 디지털 소비를 주로 하는 라이프스타일에 최적화되어 있으며, 초년도 연회비 전액 캐시백 혜택이 특징이다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 출시일 | 2024년 4월 |
| 연회비 | 국내전용 15,000원 / 해외겸용(Visa·Master) 18,000원 |
| 초년도 연회비 | 100% 캐시백 (첫 해 무료) |
| 전월실적 조건 | 30만원 이상 |
| 발급 대상 | 만 18세 이상 개인 (사회초년생 포함) |

## 주요 혜택

### 카테고리별 적립

| 카테고리 | 적립률 | 월 한도 |
|----------|--------|---------|
| 카페·편의점 | **5%** | 월 5,000 마이신한포인트 |
| 온라인 쇼핑 (쿠팡, 네이버페이, 마켓컬리 등) | **5%** | 월 5,000 마이신한포인트 |
| 정기결제 (OTT·구독 서비스) | **10~20%** | 월 4,000 마이신한포인트 |
| 그 외 일반 이용 | 0.5% | 없음 |

**정기결제 상세**

구독 서비스별 캐시백 혜택(월별 자동결제 인식 시):

| 서비스 | 적립률 |
|--------|--------|
| 넷플릭스, 유튜브 프리미엄, 왓챠 등 OTT | 20% |
| 스포티파이, 애플 뮤직, 멜론 등 음악 스트리밍 | 20% |
| 클라우드 스토리지 (iCloud, Google One) | 10% |
| 기타 월정액 구독 | 10% |

### 혜택 요약 예시 (월 소비 패턴 기준)

| 소비 항목 | 월 금액 | 적립 |
|-----------|---------|------|
| 카페·편의점 | 50,000원 | 2,500p |
| 넷플릭스 (18,500원) | 18,500원 | 3,700p |
| 쿠팡 쇼핑 | 60,000원 | 3,000p |
| 기타 | 200,000원 | 1,000p |
| **합계** | **328,500원** | **약 10,200p** |

마이신한포인트 1p = 1원 동등 가치로 사용 가능.

## 전월실적 조건

월 30만원 이상 사용 시 카테고리별 혜택 적용.

30만원 미만 사용 월에는 모든 추가 혜택 미적용, 기본 0.5% 적립만 제공.

## 마이신한포인트 활용

- 신한카드 결제 대금 차감 (1p = 1원)
- 신한은행 계좌 출금 (캐시백)
- GS25, CGV 등 제휴처 사용
- 항공 마일리지 전환 (아시아나 클럽, 대한항공 스카이패스)
- 네이버페이, 카카오페이 포인트 전환

## 경쟁 카드 비교

| 카드 | 특징 | 대상 |
|------|------|------|
| **신한카드 처음** | 카페·편의점·OTT 5~20%, 연회비 초년도 무료 | MZ·사회초년생 |
| 현대카드 ZERO3 포인트형 | 전 가맹점 1.2%, 조건 없음 | 소비 분산형 |
| KB국민카드 KB Pay | 간편결제 특화, 포인트 통합 | 모바일 결제 多 |
| 삼성카드 iD CARD | 온라인·간편결제 특화 | 디지털 소비자 |

신한카드 처음은 **OTT 구독과 카페 소비가 많은 20~30대**에게 특히 유리하다. OTT 서비스 20% 적립은 시중 카드 중 최상위 수준이다.

## 발급 팁

- 초년도 연회비 캐시백은 발급 후 익월 말일까지 한 번이라도 사용 시 자동 지급
- 해외에서도 사용하려면 Visa 또는 Mastercard 네트워크 버전(18,000원 연회비) 발급 권장
- 신한은행 계좌 보유 시 포인트를 현금으로 즉시 출금 가능해 활용도가 더 높음
- 전월 30만원 실적을 맞추기 위해 월정액 구독만으로도 일부 충당 가능

::: tip 사회초년생에게 추천하는 이유
첫 직장을 다니며 신용카드를 처음 만드는 경우, 연회비 부담 없이(초년도 무료) 일상 소비에서 가시적인 혜택을 바로 체감할 수 있는 카드다. 복잡한 조건보다는 자신이 자주 가는 카페·편의점, 쓰는 구독 서비스에서 혜택이 집중되기 때문에 추가 공부 없이도 자연스럽게 혜택을 받는다.
:::
