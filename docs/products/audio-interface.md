# 오디오 인터페이스 (Audio Interface)

오디오 인터페이스는 마이크, 기타, 신디사이저 등의 아날로그 신호를 디지털로 변환해 컴퓨터에 전달하고, 반대로 컴퓨터의 디지털 오디오를 아날로그로 변환해 스피커·헤드폰으로 출력하는 장비다. 사운드카드의 전문 외장형 버전이라고 볼 수 있다.

## 왜 오디오 인터페이스가 필요한가

컴퓨터 내장 사운드카드는 재생 품질은 충분하지만, 녹음 입력 품질이 낮고 레이턴시(지연)가 크다. 오디오 인터페이스는 다음 상황에서 필요하다.

| 상황 | 이유 |
|------|------|
| 마이크 녹음 | XLR 입력 + 팬텀파워(48V) 제공 |
| 기타·베이스 직결 | 고임피던스(Hi-Z) 입력 지원 |
| 저레이턴시 모니터링 | ASIO 드라이버로 2~5ms 이하 |
| 스피커 출력 품질 | DAC 품질이 내장보다 월등 |
| 멀티 채널 녹음 | 동시 다채널 입력 지원 |

---

## 핵심 스펙 이해

### 입력 채널 수

가장 먼저 결정해야 할 것. 동시에 몇 개의 소스를 녹음할지에 따라 선택이 달라진다.

- **2i2형 (2입력 2출력)**: 1인 유튜버·팟캐스터, 기타+보컬 동시 녹음에 충분
- **4i4형 이상**: 밴드 녹음, 드럼 마이킹, 다채널 녹음에 필요

### AD/DA 변환기 품질

아날로그↔디지털 변환기(ADC/DAC)의 품질이 음질을 결정한다. 주요 지표:

- **다이나믹 레인지**: 높을수록 좋음 (110~120dB 이상이 프로 수준)
- **THD+N**: 낮을수록 왜곡이 적음
- **샘플링 레이트**: 44.1kHz/48kHz가 기본, 96kHz/192kHz 지원 여부

### 마이크 프리앰프 (Mic Preamp)

마이크 신호는 매우 약하기 때문에 증폭이 필요하다. 프리앰프 품질이 마이크 녹음 음질에 직접적인 영향을 미친다.

- **게인 범위**: 충분한 게인(60dB 이상)이 있어야 다양한 마이크 사용 가능
- **팬텀파워 (48V)**: 콘덴서 마이크 사용 시 필수

### 레이턴시와 ASIO 드라이버

레이턴시는 소리가 마이크에서 헤드폰으로 돌아오는 지연 시간이다.

- Windows: **ASIO 드라이버** 사용 시 5ms 이하 구현 가능 (일반 MME/WDM 드라이버는 30ms 이상)
- Mac: Core Audio 기반으로 기본적으로 저레이턴시

## 주요 제품 비교

### 입문 ~ 중급 (10~20만원대)

| 제품 | 입출력 | 특징 |
|------|--------|------|
| **Focusrite Scarlett Solo (4세대)** | 1in / 2out | 가장 많이 팔리는 입문기, Air 모드 |
| **Focusrite Scarlett 2i2 (4세대)** | 2in / 2out | 입문 표준, 번들 소프트웨어 충실 |
| **PreSonus AudioBox USB 96** | 2in / 2out | 가성비, Studio One 번들 |
| **Behringer UMC22** | 2in / 2out | 최저가, 기본 성능 |

### 중급 (20~40만원대)

| 제품 | 입출력 | 특징 |
|------|--------|------|
| **SSL 2** | 2in / 2out | SSL 4K 채널 스트립 기술, 측정치 우수 |
| **SSL 2+** | 2in / 4out | SSL 2에 모니터 믹스 출력 추가 |
| **Audient iD4 Mk2** | 1in / 2out | J-FET 프리앰프, 클래스A 출력 |
| **MOTU M2** | 2in / 2out | 측정치 최상위급, 미터 디스플레이 |

### 측정치 최우수 (30~50만원대)

| 제품 | 특징 |
|------|------|
| **MOTU M4** | 4in / 4out, MOTU M2 대비 채널 확장 |
| **Focusrite Scarlett 4i4 (4세대)** | 4in / 4out, MIDI I/O 포함 |
| **Universal Audio Volt 276** | UA 1176 리미터 탑재, 하드웨어 처리 |

## Focusrite Scarlett 2i2 vs SSL 2

가장 많이 비교되는 두 제품.

| 항목 | Scarlett 2i2 4세대 | SSL 2 |
|------|--------------------|-------|
| 가격 | 약 14만원 | 약 22만원 |
| 프리앰프 | Scarlett 고유 | SSL 4K 계열 |
| 다이나믹 레인지 | ~111dB | ~113dB |
| Air 모드 | 있음 (Neve 트랜스 에뮬) | 없음 |
| 번들 소프트웨어 | Ableton Live Lite, Pro Tools 등 | Ableton Live Lite |
| 레거시 드라이버 | 있음 | ASIO4ALL 필요 (Windows) |

**선택 기준**:
- 처음 시작하는 유튜버·팟캐스터 → **Scarlett 2i2**: 드라이버 안정성, 번들 충실
- 측정치/음질 우선 → **SSL 2**: 가격 대비 프리앰프 품질 우수
- 정밀 모니터링 중요 → **MOTU M2**: 최상위 측정치

## DAC와 오디오 인터페이스의 차이

| 항목 | 오디오 인터페이스 | DAC (독립형) |
|------|-----------------|-------------|
| 주 용도 | 녹음 (입력) + 재생 | 재생 (출력) 전용 |
| XLR 마이크 입력 | 있음 | 없음 |
| 팬텀파워 | 있음 | 없음 |
| 레이턴시 최적화 | ASIO 드라이버 | 불필요 |
| 가격 (동급 기준) | 더 비쌈 | 더 저렴 |

음악 감상만이 목적이라면 오디오 인터페이스 대신 순수 DAC(헤드폰 앰프 겸용)가 합리적이다. 녹음이 필요하다면 오디오 인터페이스가 맞는 선택이다.
