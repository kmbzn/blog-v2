# CX31993 DAC 동글

CX31993은 Cirrus Logic 계열의 고성능 DAC 칩셋으로, 소형 USB-C 동글(dongle) 형태의 DAC/AMP 제품에 널리 탑재되어 있다. 소위 "꽂는 DAC"라 불리는 이 카테고리에서 중저가대 최고 성능군을 형성하며, 측정치 기반 오디오 커뮤니티(ASR, Head-Fi 등)에서 가성비 최우수 제품으로 반복적으로 언급된다.

## 칩셋 개요

| 항목 | 내용 |
|------|------|
| 제조사 | Cirrus Logic |
| 칩 계열 | CS43198 / CS43131 계열 |
| 최대 샘플링 레이트 | 384kHz / 32bit |
| 다이나믹 레인지 | 130dB 이상 (이론치) |
| SNR | ~130dB |
| THD+N | -120dB 수준 |
| 출력 임피던스 | 구현에 따라 0.1~1Ω |

CX31993 칩은 CS43198 플래그십 칩에서 파생된 더 작고 집적도 높은 버전으로, 한 칩에 DAC+AMP 기능이 통합되어 있다. 동일 칩을 사용한 제품 간에도 아날로그 출력단 설계에 따라 실측치 차이가 발생한다.

## 주요 탑재 제품

### JCALLY JM20

가장 대표적인 CX31993 동글. AliExpress에서 $10~15 수준에 구입 가능하며, 이 가격대에서 측정치가 매우 우수해 커뮤니티 추천 1순위로 자주 등장한다.

- 출력: 약 80mW @32Ω
- USB-C ↔ 3.5mm
- 크기: 매우 작음 (손가락 한 마디)
- 무게: 약 5g

### KZ AZ10 (동글형)

KZ 브랜드에서 출시한 CX31993 탑재 동글. KZ 자체 이어폰과 번들로 쓰기 좋으며, 기본 선 품질이 양호하다.

### HIDIZS S9 PRO / S9

Hidizs의 포터블 DAC 동글 라인. CS43131(CX31993 상위 호환) 탑재 모델도 있으며, 디스플레이나 볼륨 노브가 없는 심플한 구성이지만 출력이 더 강하다.

### Truthear SHIO

Truthear(이어폰 브랜드)에서 출시한 CX31993 동글. 알루미늄 하우징으로 마감이 좋고, 저임피던스·고임피던스 헤드폰 모두 무난하게 구동한다.

## 측정치 성능

ASR(Audio Science Review)에서 측정한 JCALLY JM20 기준:

| 항목 | 측정값 |
|------|--------|
| SINAD (32Ω) | 약 115dB |
| 노이즈 플로어 | 매우 낮음 |
| 출력 임피던스 | < 1Ω |
| 채널 불균형 | 무시할 수준 |

SINAD 115dB는 인간의 청력 한계(120dB SNR)에 근접한 수준으로, 음질 차이를 측정치로 판별할 수 있는 임계점을 훨씬 초과한다. 즉, 이 칩셋이 달린 제품은 음질적으로 **충분히 투명**(transparent)하다.

## 스마트폰 연결 방법

### Android (USB-C 포트)

USB-C ↔ 3.5mm 동글을 직결하면 USB Audio Class 2.0(UAC2)으로 자동 인식된다. 별도 드라이버 불필요.

- 음량 조절: 폰 볼륨 키 사용 가능 (기기에 따라 다름)
- 고해상도 오디오: Android 8.0 이상에서 USB 오디오 고해상도 지원

### iPhone (Lightning / USB-C)

- Lightning 포트 iPhone: Apple Camera Connection Kit (USB-A → Lightning) + USB-C 젠더 필요
- USB-C iPhone (iPhone 15+): 직결 가능

### PC / Mac

USB-C 또는 USB-A 젠더 경유 연결. OS 내장 USB 오디오 드라이버로 자동 인식. Windows에서는 WASAPI Exclusive 모드로 커널 믹서 우회 시 최상의 품질.

## 어떤 헤드폰·이어폰과 어울리나

CX31993 동글은 대부분 80~130mW @32Ω 수준의 출력을 가진다.

| 기기 유형 | 호환성 |
|-----------|--------|
| 대부분의 IEM (8~64Ω) | 완벽하게 구동 |
| 모바일 헤드폰 (32~80Ω) | 적합 |
| 스튜디오 헤드폰 (150~300Ω 이상) | 출력 부족 가능 |
| 평판형 헤드폰 | 모델에 따라 다름 (일반적으로 부족) |

Sennheiser HD650 (300Ω), AKG K702 같은 고임피던스 헤드폰을 최대 볼륨으로 쓰고 싶다면 별도 데스크탑 앰프가 필요하다.

## 비교: 스마트폰 내장 DAC vs CX31993 동글

| 항목 | 스마트폰 내장 | CX31993 동글 |
|------|--------------|--------------|
| 다이나믹 레인지 | ~90~105dB | ~115dB |
| 노이즈 플로어 | 높은 편 (그라운드 루프, 전자기 간섭) | 낮음 |
| 임피던스 매칭 | 부정확한 경우 많음 | 정확 |
| 출력 | 모델마다 차이 큼 | 일관됨 |
| 가격 (추가 비용) | 0 | 1~3만원 |

고급 IEM이나 헤드폰을 쓰면서 내장 DAC의 한계(배경 히스, 임피던스 불일치로 인한 주파수 응답 변화)를 느끼고 있다면 CX31993 동글로 충분히 해결된다.
