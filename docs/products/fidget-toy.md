# 피젯 토이 (Fidget Toy)

피젯 토이(Fidget Toy)는 손으로 만지고 조작하면서 불안이나 무료함을 해소하고 집중력을 유지하는 데 도움을 주는 소형 장난감이다. "피젯(fidget)"은 영어로 "안절부절못하다", "손발을 가만히 두지 못하다"는 뜻이다. 특히 ADHD, 불안 장애, 자폐 스펙트럼을 가진 사람들의 감각 조절 도구로 주목받았으며, 일반인에게는 스트레스 해소와 습관적 손 놀림(펜 돌리기, 손가락 두드리기 등)을 대체하는 용도로 쓰인다.

---

## 주요 종류

### 피젯 스피너 (Fidget Spinner)

2017년 전 세계적으로 폭발적인 유행을 일으킨 피젯 토이. 중앙 베어링을 엄지와 중지로 잡고 검지로 튕기면 날개가 수분간 돌아간다.

- **구조**: 중앙 베어링(ABEC-7 이상) + 3~5개 날개(또는 2개)
- **소재**: ABS 플라스틱 / 황동 / 스테인리스 스틸 / 티타늄
- **특징**: 회전 유지 시간(spin time)이 스펙. 고급 금속 베어링 사용 시 5~10분 이상 유지
- **유행**: 2017년 초 SNS에서 바이럴 → 전 세계 판매량 수억 개
- **하락**: 2017년 하반기부터 급격히 식었으나 매니아 층은 유지

| 소재 | 특징 | 가격대 |
|------|------|--------|
| 플라스틱 | 가벼움, 저렴 | 1,000~5,000원 |
| 황동/구리 | 묵직한 무게감, 긴 스핀타임 | 1~5만원 |
| 티타늄 | 경량 + 고강도, 최상급 | 5~20만원 |
| 스틸 | 균형 잡힌 무게 | 2~8만원 |

### 피젯 큐브 (Fidget Cube)

2016년 Antsy Labs가 킥스타터를 통해 선보인 6면체 피젯 장난감. 각 면에 서로 다른 조작 방식이 있어 다양한 자극을 제공한다.

**6개 면 구성**

| 면 | 기능 |
|----|------|
| 클릭 | 5개 버튼 (3개는 클릭감, 2개는 무소음) |
| 글라이드 | 조이스틱 형태의 구슬 |
| 플립 | 딸깍거리는 스위치 |
| 브리드 | 누르고 있는 홈 (단순 압박감) |
| 롤 | 기어 + 회전볼 |
| 스핀 | 회전 다이얼 |

- 원본 Antsy Labs 제품 외 수많은 유사품이 유통됨
- 키체인 크기의 미니 버전도 있음

### 팝 잇 (Pop It)

2020~2021년 코로나 시기에 전 세계적으로 유행한 실리콘 버블 피젯 토이. 뽁뽁이(에어캡) 누르는 감촉을 무한히 재현할 수 있다.

- 실리콘 소재의 돌기를 누르면 "딸깍" 소리 + 촉감 → 반대편에서 다시 누르면 원상복구
- 다양한 형태(하트, 원, 공룡, 과일 등)와 컬러
- 2인 게임 방식 있음 (번갈아 한 번에 원하는 수만큼 누름, 마지막 남은 칸을 누르는 사람이 짐)
- 세척 가능, 반영구 사용

### 인피니티 큐브 (Infinity Cube)

접히는 마디들이 무한히 변형되는 구조의 피젯 토이. 정육면체 형태에서 시작해 손목 스냅으로 접고 펴는 동작을 반복한다. 단순하지만 손에서 내려놓기 어려운 특성이 있어 회의 중이나 집중 작업 중 무의식적으로 손이 가는 형태다.

- 소재: ABS 플라스틱 / 알루미늄
- 크기: 2.5~4cm 수준으로 휴대성 좋음

### 마그네틱 링 / 마그네틱 볼 (Magnetic Fidget)

강력한 네오디뮴 자석으로 만든 피젯 도구들.

- **마그네틱 링**: 손가락 사이에서 회전하며 딸깍거림. 고급 버전은 티타늄 또는 스테인리스 스틸
- **마그네틱 볼 (M-Ball, Newton's cradle 계열)**: 두 손에 하나씩 쥐고 서로 가까이 가져가면 자력으로 달라붙고 튕기는 촉감
- **주의**: 강력 자석은 어린이가 삼키면 위험. 어른용 명시 제품 권장

### 슬라임 / 킨에틱 샌드 (Kinetic Sand)

감각 자극 피젯의 일종.

- **슬라임**: 폴리비닐알코올(PVA) 풀 + 붕사/콘택트렌즈 세정제 혼합으로 만든 점탄성 소재. 늘리고 찌르는 촉감이 특징
- **킨에틱 샌드**: 모래 97% + 실리콘 폴리머 3%. 젖은 모래처럼 뭉치지만 실제로는 건식. 형태 유지와 흘러내림이 공존하는 독특한 촉감

## 효과에 대한 연구

피젯 토이의 집중력 향상 효과는 연구 결과가 엇갈린다.

**긍정적 연구**
- ADHD 아동에서 운동 자극(fidgeting)이 인지 과제 수행 시 도움이 된다는 연구 있음
- 반복적인 손 움직임이 코르티솔(스트레스 호르몬) 감소에 기여한다는 연구 있음
- 감각 처리 이상(자폐 스펙트럼 등)에서 자기 조절 도구로 유효하다는 임상 보고

**부정적·중립적 연구**
- 일반 초등학생 대상 연구에서 피젯 스피너가 수업 집중력을 **저하**시킨다는 결과
- 많은 효과 주장이 일화적(anecdotal) 근거에 머무름
- 대규모 임상 연구 부족

**현재 컨센서스**: ADHD 진단을 받은 사람, 또는 감각 처리에 어려움이 있는 사람에게는 도움이 될 수 있으나, 일반적인 집중력 향상 도구로서의 효과는 과학적으로 충분히 입증되지 않았다.
